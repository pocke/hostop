hostop
=======

Stop accessing to noisy host


Installation
---------


```bash
go get github.com/pocke/hostop
```


Usage
--------


```bash
$ hostop twitter.com --after 8h
```

Known Issue
------

You should restart your browser to apply the stopping. Maybe a browser caches DNS.
